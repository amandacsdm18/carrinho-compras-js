## 🛒 Carrinho de Compras em JavaScript

Projeto desenvolvido com o objetivo de simular um carrinho de compras utilizando JavaScript.

## 🚀 Funcionalidades
- Adicionar produtos
- Remover produtos
- Cálculo automático do total

## 💻 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 🎯 Objetivo
Praticar lógica de programação e desenvolvimento web, aplicando conceitos aprendidos no curso de Análise e Desenvolvimento de Sistemas.
